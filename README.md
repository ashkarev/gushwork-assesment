# Gushwork Frontend Assignment

A responsive frontend implementation built using vanilla HTML, CSS, and JavaScript based on the provided Figma design.

## Features

* Responsive layout for desktop, tablet, and mobile
* Sticky header with smooth scroll behavior
* Interactive image carousel
* Hover zoom functionality for carousel images
* Smooth transitions and animations
* Semantic HTML structure
* Optimized and organized CSS
* Cross-browser compatible implementation

## Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla)

## Project Structure

```bash
├── index.html
├── styles.css
├── script.js
├── assets/
│   ├── images/
│   └── fonts/
└── README.md
```

## Getting Started

1. Clone the repository

```bash
git clone 
```

2. Open the project folder

3. Run `index.html` in your browser
